きこりちゃん

より多くの木を切って世界中の人とスコアを競うゲーム

UnityC#2D

概要と目的

本プロジェクトは、「指一本でもプレイできる」をコンセプトにしたハイパーカジュアルゲームです。
プレイヤーは左右矢印キーのみを使い、主人公となる「きこりちゃん」を操作してより多くの木を切りスコアを競います。
また本ゲームはスマートフォンにも対応しており、感覚で操作できるようタップ操作を採用しています。

主な役割と技術

  • ゲームプレイプログラミング

    Unity C#を用いてゲーム内ロジックをすべて担当。自身が後から設計を拡張しやすいように各機能を分割しながら開発を進めた。

  • イラスト

    Asepriteを用いてゲーム内のイラストをすべて担当。プレイする人が見やすく、かつ印象に残るよう各イラストの配置や色遣いに気を付けながら制作を行った。

  • UI

    Aseprite,Adobe Illustratorを用いてゲーム内UIをすべて担当。シンプルかつ見やすいデザインを意識して制作を行った。

  • 開発環境

    Unity6.0.59f2
    VSCode
    Git
    Aseprite
    Adobe Illusrator

  • 政策背景

    指一本でもできるゲームをコンセプトに一つの作品を作ろうと思い作成。

  • 課題と学び

    作成したイラストをUnityに持っていく際にサイズを調節することが最大の課題だった。ここはUnityの機能であるPixel Perfect Cameraを利用することでSprite側を拡大せず、Unityに自動補完させることでイラストを崩さずに持ってくることが可能になった。
    また、ランキング機能においてはサーバを用意することが難しかったため、unityroomを利用し実装を行った。

技術詳細

  • Object Poolingを導入し、過度に行われるインスタンスの生成、破棄の処理を抑制。GC発生を削減し、処理負荷の最適化を行った。
  • Unityの標準機能であるPixel Perfect Cameraを利用しSpriteを破綻させずに処理を可能にした。
  • Aseprite, Illustratorなどを利用しUIやイラストなどの作成を行った。
